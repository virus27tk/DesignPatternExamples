# ParkingLot

A .NET Console Application for managing a Parking Lot system.

## Project Structure

- **Program.cs**: Entry point for the application
- **Models/**: Contains data models (e.g., ParkingLot, Vehicle)
- **Services/**: Business logic and service classes

## Getting Started

### Prerequisites
- [.NET 6.0 SDK or later](https://dotnet.microsoft.com/download)

### Build and Run

1. Open a terminal in the project root directory.
2. Run the following commands:

   ```sh
   dotnet build
   dotnet run
   ```

## Features
- Basic ParkingLot class
- Simple menu-driven console interface

## Extending
- Add new models to the `Models` folder
- Add new business logic to the `Services` folder
